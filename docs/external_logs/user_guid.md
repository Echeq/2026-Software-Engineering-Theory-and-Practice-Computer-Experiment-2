# User Guide - SPMP Platform

## Getting Started

### Accessing the Application
1. Open your web browser
2. Navigate to `http://localhost:8080`
3. You will see the login page

### Creating an Account
1. Click "Sign Up" on the login page
2. Fill in the registration form:
   - Full Name: Your complete name
   - Email: A valid email address
   - Password: At least 6 characters
   - Confirm Password: Must match your password
3. Click "Sign Up"
4. You will be redirected to the login page

### Logging In
1. Enter your registered email address
2. Enter your password
3. Click "Log In"
4. You will be redirected to the dashboard

## Dashboard Features

### Managing Projects

#### Creating a Project
1. Click the "+ New Project" button
2. Enter a project name (required)
3. Optionally add a description
4. Click "Create Project"

#### Viewing Projects
- All your projects are displayed as cards on the dashboard
- Each card shows the project name, description, and status

#### Deleting a Project
1. Click the "Delete" button on the project card
2. Confirm the deletion
3. Note: This will also delete all tasks in the project

### Managing Tasks

#### Adding a Task
1. Click "+ Add Task" on a project card
2. Fill in the task details:
   - Task Title (required)
   - Description (optional)
   - Priority: Low, Medium, or High
   - Due Date (optional)
3. Click "Add Task"

#### Updating Task Status
1. Locate the task in the "My Tasks" section
2. Use the status dropdown to change:
   - Pending
   - In Progress
   - Completed

#### Viewing Tasks
- Tasks assigned to you appear in the "My Tasks" section
- Each task shows title, description, priority badge, and due date

## Tips
- Keep your password secure
- Use descriptive project names and task titles
- Set priorities to manage your workload effectively
- Regularly update task statuses to track progress

## Troubleshooting
- If login fails, check your email and password
- Ensure you're connected to the server at localhost:8080
- Clear browser cache if pages don't load correctly
